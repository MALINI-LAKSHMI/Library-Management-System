# Library-Management-System

Description:

This project is a Library Management System developed using Java, JDBC, and Apache Tomcat.
The main aim of this project is to reduce manual work in libraries and make book management easy.
It helps in maintaining book details, user details, and issuing and returning books in a simple way.

This project is built as a Java web application and connected to a MySQL database using JDBC.
